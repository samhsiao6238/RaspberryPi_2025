# 進程管理指令

<br>

## 說明

1. `top`：顯示系統進程與資源使用情況。

    ```bash
    top
    ```

<br>

2. `ps`：列出當前進程。

    ```bash
    # `-e` 顯示所有進程，即使不是由當前用戶啟動的進程
    # `-f`顯示進程完整訊息，包括 UID、PID、PPID、C、STIME、TTY、TIME 和 CMD 等欄位
    ps -ef
    ```

<br>

3. `kill`：終止進程。

    ```bash
    kill -9 <進程ID>
    ```
