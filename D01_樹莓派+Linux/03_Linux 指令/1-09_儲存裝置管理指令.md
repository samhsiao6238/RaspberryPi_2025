# 儲存裝置管理指令

<br>

## 儲存空間查詢與掛載

1. `df`：顯示磁碟空間使用狀況。

    ```bash
    # `-h` 以人類可讀格式顯示磁碟使用情況
    df -h
    ```

<br>

2. `du`：顯示目錄或檔案的大小。

    ```bash
    # `-s` 總結顯示指定目錄或檔案的大小
    # `-h` 以人類可讀格式顯示大小
    du -sh <目錄或檔案名稱>
    ```

<br>

3. `lsblk`：列出所有區塊裝置，如硬碟、分割區、SD 卡；`-f` 顯示檔案系統資訊，如掛載點與 UUID。

    ```bash
    lsblk -f
    ```

<br>

4. `mount`：顯示當前掛載的檔案系統。

    ```bash
    mount | column -t
    ```

<br>

5. `findmnt`：以階層結構顯示掛載點，為 `mount` 的進階版。

    ```bash
    findmnt
    ```

<br>

6. `blkid`：查詢所有儲存裝置的 UUID、檔案系統類型等識別資訊。

    ```bash
    sudo blkid
    ```

<br>

7. `df -T`：顯示磁碟空間同時列出檔案系統格式類型。

    ```bash
    df -Th
    ```

<br>
