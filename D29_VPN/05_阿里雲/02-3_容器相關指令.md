# 容器相關指令

_補充說明，無需實作_

<br>

1. 停止容器。

    ```bash
    docker stop openvpn-as
    ```

<br>

2. 使用參數 `rm` 移除容器。

    ```bash
    docker rm openvpn-as
    ```

<br>

3. 使用參數 `rmi` 移除包含鏡像在內的全部容器文件。

    ```bash
    docker rmi openvpn/openvpn-as
    ```

<br>

4. 重啟容器。

    ```bash
    docker restart openvpn-as
    ```

<br>
