# 重啟 EC2

## 步驟

1. 重啟

![](images/img_46.png)

2. 進入本機終端機

```bash
cd ~/Downloads && ls *.ovpn && rm -f *.ovpn && ls *.pem
```

3. 點擊 `Connect`

4. 複製連線指令，並將 `root` 改為 `openvpnas`

![](images/img_47.png)

5. 連線後沒有像第一次登入時自動進入設定頁面，

![](images/img_48.png)

## 根據指示操作

1. 切換

```bash
sudo -i
```

2. 安裝

```bash
ovpn-init
```

3. 輸入 `DELETE`；接下來就跟原本的設定一樣，不再贅述。

![](images/img_49.png)

4. 直到出現設定密碼的時候，再次輸入要設定的密碼

    ![](images/img_50.png)

5. 依據新的鏈接進行後續步驟，使用者名稱。

![](images/img_51.png)
